## The Build Context

Skissify is a REST API that converts JSON into hand-drawn SVG sketches. I built it because Claude kept reasoning about spatial layouts — floor plans, diagrams, architecture sketches — but had no way to output them visually.

Text descriptions of visual things are a lossy format. "A rectangular room with a door on the north wall" is not the same as seeing it.

So I built the drawing tool. Now Claude draws.

The MCP server (`npx skissify-mcp`) makes it a 30-second install in Claude Desktop. After that, you can ask Claude to sketch your apartment and it will.
