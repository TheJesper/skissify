## What Skissify Is (Quick Version)

A tool that renders JSON as hand-drawn sketches. Floor plans, diagrams, wireframes.

The key feature: AI agents can use it natively via MCP. Tell Claude to design something, it writes JSON, Skissify draws it. No drag-and-drop. No human in the loop.

Free tier at skissify.com. Pro at €5/month.
