## Platform-Specific Notes

### TikTok
- Post at 7-9am or 7-9pm (Stockholm time → also hits US morning/afternoon)
- Use the #AIGotHands tag before it gets saturated — we're claiming it now
- Hook in first 2 seconds or they swipe. Text on screen from frame 1.
- Reply to every comment in the first hour — algorithm boost

### Instagram Reels
- Same video, different caption tone (slightly more lifestyle/aspirational)
- Tag: @anthropic, @claude.ai if possible
- Add to Stories with "New post" sticker to drive views

### YouTube Shorts
- Title: "I gave Claude a pencil and it drew me a floor plan"
- Add to Skissify channel playlist immediately
- YouTube Shorts SEO: "AI floor plan generator", "Claude draws", "MCP tools 2026"

### Twitter/X
- Post as embedded video, not link
- First tweet in thread = video. Second tweet = technical details.
- Replying to your own thread with "Source code / API docs:" increases views
