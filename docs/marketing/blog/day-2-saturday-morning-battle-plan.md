## Human Mode Is Moving Up

I planned Human Mode (natural language → sketch, no JSON) for Week 2.

After seeing 40%+ of signups come from non-developers, it's moving to this weekend.

The workflow that's working without it:
1. User describes room to Claude
2. Claude generates Skissify JSON
3. User pastes JSON to editor
4. Gets a floor plan

The workflow with Human Mode:
1. User describes room in a text box
2. Skissify calls Claude API
3. Gets a floor plan

Same outcome. One step instead of three. No JSON visible.

The technical work is maybe 4-6 hours. I'm doing it today.

| Model | First-try success |
|-------|-----------------|
| Claude Sonnet 4 | 94% |

The gap between Claude and Llama 3B isn't about schema knowledge. It's about spatial reasoning. Small models don't understand that a door needs to intersect a wall.
