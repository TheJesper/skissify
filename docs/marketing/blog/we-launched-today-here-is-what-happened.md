## What Is Skissify (30-Second Version)

Skissify turns JSON into hand-drawn sketches. The input is not a mouse — it's structured data. Which means AI agents can generate sketches programmatically.

Install the MCP server, tell Claude to "design a floor plan," and you get back an actual sketch — not text describing a floor plan. An image. A hand-drawn-looking floor plan with wobbling walls and little pencil textures.

That's the thing. That's what I built.
