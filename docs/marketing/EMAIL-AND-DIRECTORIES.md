## Launch Announcement Email

**Subject lines (A/B test):**
- A: "Skissify is live — sketches from JSON, built for AI agents"
- B: "I just launched a sketch tool that AI agents can use"
- C: "JSON in, hand-drawn sketches out — Skissify is here"

**Email body:**

```
Hi [name],

I just launched Skissify — a web tool that turns JSON into hand-drawn sketches.

Here's the 30-second version:

- Paste JSON describing elements (lines, rectangles, doors, windows, text)
- Get a hand-drawn-style rendering instantly
- Adjust wobble, paper type, and "humanness"
- AI agents can create sketches via MCP (Model Context Protocol)

Why this exists: AI agents can generate text, code, and data — but they couldn't create sketches. Now they can. Skissify has an MCP server that lets Claude, GPT, and other AI assistants generate floor plans, diagrams, and wireframes from natural language.

Try it free: https://skissify.com (no signup needed for the editor)

It's free for public sketches. Pro at EUR 5/month adds private sketches, sharing, and API access.

I'd love to know what you think. Reply to this email or find me on Twitter @[handle].

Best,
[name]
```
