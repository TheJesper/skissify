## Maker's First Comment

Post this immediately after the listing goes live:

```
Hey Product Hunt! 👋

I'm [name], and I built Skissify because I wanted AI agents to be able to create sketches.

The idea is simple: you describe a sketch in JSON (or let an AI agent do it), and Skissify renders it as a hand-drawn-style image. Think floor plans, architecture diagrams, wireframes — anything you'd sketch on a napkin.

Why JSON? Because it means any AI agent can create sketches programmatically. Skissify has an MCP server, so Claude and other AI assistants can generate sketches directly. No drag-and-drop needed.

The wobble engine uses multi-harmonic algorithms to make lines look authentically hand-drawn — not the simple S-curve you see in other tools. You can tune the amplitude, frequency, and "humanness" to go from clean technical drawings to messy napkin sketches.

What's here today:
- 14 element types (including architectural: doors, windows, stairs, dimension lines)
- 4 paper styles (cream, white, grid, blueprint)
- 3 tool types (pencil, ballpoint, ink)
- MCP server for AI agent integration
- Free tier with public sketches
- Pro at €5/mo for private sketches + API

I'd love to hear what you'd use it for. Floor plans? System diagrams? Wireframes? What element types should I add next?
```
